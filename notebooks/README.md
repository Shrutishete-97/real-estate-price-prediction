# Real Estate Price Prediction

## Objective
Build a machine learning model to predict residential property prices based on property features.

## Dataset
The dataset includes features such as:
- BHK
- Area
- Age
- Property Type
- Locality
- Region
- Status

## Approach
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- One-Hot Encoding for categorical variables
- Linear Regression model training
- Model evaluation using R² and MAE

## Results
- R² Score: ~0.82
- MAE: ~43

## Tools & Technologies
- Python
- Pandas, NumPy
- Seaborn, Matplotlib
- Scikit-learn

## Conclusion
The model successfully captures key factors influencing property prices and provides reliable predictions.
